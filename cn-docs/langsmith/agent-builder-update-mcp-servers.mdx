<Warning>
更改自定义 MCP 服务器的 URL 将导致使用该服务器工具的智能体失效。
</Warning>

Agent Builder 通过 MCP 服务器 URL 存储工具引用。如果您更新了自定义 MCP 服务器的 URL，现有智能体在尝试调用这些工具时将失败，因为存储的 URL 不再匹配。

## 如何更新 MCP 服务器 URL

1.  在工作区设置中更新您的 MCP 服务器 URL
2.  对于每个使用该服务器工具的智能体：
    *   从智能体配置中移除受影响的工具
    *   重新添加这些工具（它们现在将引用新的 URL）
3.  测试智能体以确认工具正常工作
