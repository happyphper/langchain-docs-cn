**LangSmith 部署** 基于开源的 [LangGraph](/oss/langgraph/overview) 框架构建，用于开发有状态的多智能体应用。
LangGraph 提供了核心抽象和执行模型，而 LangSmith 则增加了托管基础设施、可观测性、部署选项、助手和并发控制——支持从开发到生产的完整生命周期。

<Callout icon="cubes" color="#4F46E5" iconType="regular">
LangSmith 部署是框架无关的：您可以部署使用 LangGraph 或[其他框架](/langsmith/deploy-other-frameworks)构建的智能体。要开始使用 LangGraph 本身，请参阅 [LangGraph 快速入门](/oss/langgraph/quickstart)。
</Callout>
