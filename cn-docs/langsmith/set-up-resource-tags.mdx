<Info>
资源标签功能适用于 Plus 和 Enterprise 计划。
</Info>

工作空间有助于分离信任边界和访问控制，而标签则帮助您在工作空间内组织资源。标签是您可以附加到资源上的键值对。

<Note>
**请勿与提交标签混淆**：资源标签是用于组织和筛选工作空间资源（项目、数据集、提示词等）的键值对。[提交标签](/langsmith/manage-prompts#commit-tags) 是引用提示词提交历史中特定版本的标签。虽然这两种标签可能使用相似的术语（如 `prod` 或 `staging`），但资源标签帮助您在工作空间内*组织资源*，而提交标签则控制代码中使用的提示词的*版本*。
</Note>

## 创建标签

要创建标签，请前往工作空间设置，点击“资源标签”选项卡。在这里，您可以看到按键分组的现有标签值。默认会创建两个键 `Application` 和 `Environment`；`Application` 键用于筛选界面中显示的资源。

要创建新标签，请点击“新建标签”按钮。系统会提示您输入标签的键和值。请注意，您可以使用现有的键或创建一个新的。

![创建标签](/langsmith/images/create-tag.png)

## 为资源分配标签

在创建新标签的同一侧边栏面板中，您也可以将资源分配给标签。在“分配资源”部分搜索相应的资源，并选择您想要标记的资源。

<Note>
您只能使用资源标签标记工作空间范围内的资源。这包括追踪项目、标注队列、部署、实验、数据集和提示词。
</Note>

要从资源中移除标签，可以在标签面板和资源标签面板中，点击标签旁边的垃圾桶图标。

## 删除标签

您可以从[工作空间设置页面](https://smith.langchain.com/settings/workspaces/resource_tags)删除标签的键或值。要删除键，请点击键旁边的垃圾桶图标。要删除值，请点击值旁边的垃圾桶图标。

请注意，如果您删除一个键，与该键关联的所有值也将被删除。当您删除一个值时，您将失去该值与资源之间的所有关联。

![删除标签](/langsmith/images/delete-tag.png)
