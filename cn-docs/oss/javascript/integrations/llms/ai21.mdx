您可以通过[在 AI21Labs 网站上注册](https://www.ai21.com/)获取 API 密钥，开始使用 AI21Labs 的 Jurassic 系列模型，并查看所有可用的基础模型完整列表。

以下是在 LangChain.js 中初始化实例的示例：

<Tip>
有关安装 LangChain 包的通用说明，请参阅[此部分](/oss/langchain/install)。
</Tip>

```bash npm
npm install @langchain/community @langchain/core
```

```typescript
import { AI21 } from "@langchain/community/llms/ai21";

const model = new AI21({
  ai21ApiKey: "YOUR_AI21_API_KEY", // 或设置为 process.env.AI21_API_KEY
});

const res = await model.invoke(`Translate "I love programming" into German.`);

console.log({ res });

/*
  {
    res: "\nIch liebe das Programmieren."
  }
 */
```
