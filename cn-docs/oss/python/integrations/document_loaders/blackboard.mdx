>[Blackboard Learn](https://en.wikipedia.org/wiki/Blackboard_Learn)（前身为 Blackboard 学习管理系统）是一个基于网络的虚拟学习环境和学习管理系统，由 Blackboard Inc. 开发。该软件具有课程管理、可定制的开放架构和可扩展的设计，允许与学生信息系统和身份验证协议集成。它可以安装在本地服务器上，由 `Blackboard ASP Solutions` 托管，或作为软件即服务（SaaS）在亚马逊网络服务（AWS）上提供。其主要用途包括为传统面对面授课的课程增加在线元素，以及开发完全在线、极少或无需面对面会议的课程。

本文档介绍了如何从 [Blackboard Learn](https://www.anthology.com/products/teaching-and-learning/learning-effectiveness/blackboard-learn) 实例加载数据。

此加载器并非兼容所有 `Blackboard` 课程。它仅兼容使用新版 `Blackboard` 界面的课程。
要使用此加载器，您必须拥有 BbRouter cookie。您可以通过登录课程，然后从浏览器的开发者工具中复制 BbRouter cookie 的值来获取此 cookie。

```python
from langchain_community.document_loaders import BlackboardLoader

loader = BlackboardLoader(
    blackboard_course_url="https://blackboard.example.com/webapps/blackboard/execute/announcement?method=search&context=course_entry&course_id=_123456_1",
    bbrouter="expires:12345...",
    load_all_recursively=True,
)
documents = loader.load()
```
