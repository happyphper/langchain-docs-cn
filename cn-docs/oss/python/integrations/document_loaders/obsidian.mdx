>[Obsidian](https://obsidian.md/) 是一个强大且可扩展的知识库，它基于您本地的纯文本文件文件夹工作。

本笔记本介绍了如何从 `Obsidian` 数据库中加载文档。

由于 `Obsidian` 在磁盘上仅存储为一个包含 Markdown 文件的文件夹，因此加载器只需接收该目录的路径即可。

`Obsidian` 文件有时还包含[元数据](https://help.obsidian.md/Editing+and+formatting/Metadata)，这是文件顶部的一个 YAML 块。这些值将被添加到文档的元数据中。（`ObsidianLoader` 也可以传入 `collect_metadata=False` 参数来禁用此行为。）

```python
from langchain_community.document_loaders import ObsidianLoader
```

```python
loader = ObsidianLoader("<path-to-obsidian>")
```

```python
docs = loader.load()
```
