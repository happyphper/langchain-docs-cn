---
title: KoboldAI API
---
[KoboldAI](https://github.com/KoboldAI/KoboldAI-Client) 是一个“基于浏览器的前端，用于 AI 辅助写作，支持多种本地和远程 AI 模型...”。它提供了一个公共和本地 API，可以在 LangChain 中使用。

本示例将介绍如何将 LangChain 与该 API 结合使用。

文档可以在浏览器中通过在你的端点后添加 `/api` 来找到（例如 [127.0.0.1/:5000/api](http://127.0.0.1/:5000/api)）。

```python
from langchain_community.llms import KoboldApiLLM
```

将下面看到的端点替换为使用 `--api` 或 `--public-api` 启动 webui 后输出中显示的端点。

可选地，你可以传入像 `temperature` 或 `max_length` 这样的参数。

```python
llm = KoboldApiLLM(endpoint="http://192.168.1.144:5000", max_length=80)
```

```python
response = llm.invoke(
    "### Instruction:\nWhat is the first book of the bible?\n### Response:"
)
```
