> [ClearML](https://github.com/allegroai/clearml) 是一个 ML/DL 开发与生产套件，包含 5 个主要模块：
>
> - `Experiment Manager` - 自动化的实验跟踪、环境和结果管理
> - `MLOps` - 面向 ML/DL 作业（K8s / 云 / 裸机）的编排、自动化与流水线解决方案
> - `Data-Management` - 基于对象存储（S3 / GS / Azure / NAS）的完全可微分数据管理与版本控制解决方案
> - `Model-Serving` - 云就绪的可扩展模型服务解决方案！
    在 5 分钟内部署新的模型端点
    包含由 Nvidia-Triton 支持的优化 GPU 服务
    提供开箱即用的模型监控
> - `Fire Reports` - 创建和分享支持嵌入式在线内容的丰富 MarkDown 文档

为了妥善跟踪您的 LangChain 实验及其结果，您可以启用 `ClearML` 集成。我们使用 `ClearML Experiment Manager` 来整洁地跟踪和组织您的所有实验运行。

## 回调函数

```python
from langchain_community.callbacks import ClearMLCallbackHandler
```

```python
from langchain_core.callbacks import StdOutCallbackHandler
from langchain_openai import OpenAI

# 设置并使用 ClearML 回调
clearml_callback = ClearMLCallbackHandler(
    task_type="inference",
    project_name="langchain_callback_demo",
    task_name="llm",
    tags=["test"],
    # 根据您希望跟踪的详细程度更改以下参数
    visualize=True,
    complexity_metrics=True,
    stream_logs=True,
)
callbacks = [StdOutCallbackHandler(), clearml_callback]
# 准备 OpenAI 模型
llm = OpenAI(temperature=0, callbacks=callbacks)
```

```text
clearml 回调目前处于测试阶段，可能会根据 `langchain` 的更新而更改。如有任何问题，请报告至 https://github.com/allegroai/clearml/issues 并标记 `langchain`。
```
