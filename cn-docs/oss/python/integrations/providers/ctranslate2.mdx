>[CTranslate2](https://opennmt.net/CTranslate2/quickstart.html) 是一个用于对 Transformer 模型进行高效推理的 C++ 和 Python 库。

该项目实现了一个自定义运行时，应用了多种性能优化技术，例如权重量化、层融合、批处理重排序等，以在 CPU 和 GPU 上加速并减少 Transformer 模型的内存使用。

完整的功能列表和受支持的模型包含在[项目仓库](https://opennmt.net/CTranslate2/guides/transformers.html)中。要开始使用，请查看官方的[快速入门指南](https://opennmt.net/CTranslate2/quickstart.html)。

## 安装与设置

安装 Python 包：

<CodeGroup>
```bash pip
pip install ctranslate2
```

```bash uv
uv add ctranslate2
```
</CodeGroup>
