>[Datadog](https://www.datadoghq.com/) 是一个面向云规模应用的监控与分析平台。

## 安装与设置

<CodeGroup>
```bash pip
pip install datadog_api_client
```

```bash uv
uv add datadog_api_client
```
</CodeGroup>

我们必须使用 Datadog API 密钥和应用密钥来初始化加载器，并且需要设置查询以提取所需的日志。

## 文档加载器

查看[使用示例](/oss/integrations/document_loaders/datadog_logs)。

from langchain_community.document_loaders import DatadogLogsLoader
