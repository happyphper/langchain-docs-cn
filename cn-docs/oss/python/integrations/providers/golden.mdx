>[Golden](https://golden.com) 提供了一套自然语言 API，用于通过 Golden 知识图谱进行查询和丰富化，例如：可以使用诸如 `Products from OpenAI`、`Generative ai companies with series a funding` 和 `rappers who invest` 等查询来检索相关实体的结构化数据。
>
>`golden-query` langchain 工具是 [Golden Query API](https://docs.golden.com/reference/query-api) 的一个封装，它支持以编程方式访问这些结果。
>更多信息请参阅 [Golden Query API 文档](https://docs.golden.com/reference/query-api)。

## 安装与设置
- 前往 [Golden API 文档](https://docs.golden.com/) 以了解 Golden API 的概览。
- 从 [Golden API 设置](https://golden.com/settings/api) 页面获取您的 API 密钥。
- 将您的 API 密钥保存到 GOLDEN_API_KEY 环境变量中。

## 封装器

### 实用工具

存在一个封装此 API 的 GoldenQueryAPIWrapper 实用工具。导入此实用工具：

```python
from langchain_community.utilities.golden_query import GoldenQueryAPIWrapper
```

关于此封装器的更详细演练，请参阅 [此笔记本](/oss/integrations/tools/golden_query)。

### 工具

您也可以轻松地将此封装器作为工具加载（以便与代理一起使用）。
您可以通过以下方式实现：
```python
from langchain_community.agent_toolkits.load_tools import load_tools
tools = load_tools(["golden-query"])
```

有关工具的更多信息，请参阅 [此页面](/oss/integrations/tools)。
