title: Lindorm
---
Lindorm 是阿里云推出的云原生多模数据库，它支持宽表、时序、文本、对象、流、空间等多种数据的统一接入与融合处理。它兼容 SQL、HBase/Cassandra/S3、TSDB、HDFS、Solr、Kafka 等多种标准接口，并能与第三方生态工具无缝集成。这使得它适用于日志、监控、计费、广告、社交、出行、风控等多种场景。Lindorm 也是支撑阿里核心业务的数据库之一。

要使用 Lindorm 的 AI 和向量能力，您需要[获取服务](https://help.aliyun.com/document_detail/174640.html?spm=a2c4g.11186623.help-menu-172543.d_0_1_0.4c6367558DN8Uq)并安装 `langchain-lindorm-integration` 包。

```python
!pip install -U langchain-lindorm-integration
```

## 嵌入模型

要使用部署在 Lindorm AI 服务中的嵌入模型，请导入 `LindormAIEmbeddings`。

```python
from langchain_lindorm_integration import LindormAIEmbeddings
```

## 重排序

Lindorm AI 服务也支持重排序功能。

```python
from langchain_lindorm_integration.reranker import LindormAIRerank
```

## 向量存储

Lindorm 也支持向量存储。

```python
from langchain_lindorm_integration import LindormVectorStore
```

## 字节存储

使用 Lindorm 的字节存储功能。

```python
from langchain_lindorm_integration import LindormByteStore
```
