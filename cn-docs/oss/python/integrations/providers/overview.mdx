LangChain 提供了一个广泛的生态系统，拥有 1000 多个集成，涵盖聊天与嵌入模型、工具与工具包、文档加载器、向量存储等。

**提供商**（provider）是 LangChain 集成的第三方服务或平台，用于访问聊天模型、嵌入和向量存储等 AI 能力。这些提供商拥有独立的 `langchain-provider` 包，以实现更好的版本控制、依赖管理和测试。
