title: PipelineAI
---
本页介绍了如何在 LangChain 中使用 PipelineAI 生态系统。
内容分为两部分：安装与设置，以及特定 PipelineAI 封装器的参考。

## 安装与设置

- 使用 `pip install pipeline-ai` 安装
- 获取 Pipeline Cloud API 密钥，并将其设置为环境变量 (`PIPELINE_API_KEY`)

## 封装器

### LLM

LangChain 提供了一个 PipelineAI LLM 封装器，您可以通过以下方式访问：

```python
from langchain_community.llms import PipelineAI
```
