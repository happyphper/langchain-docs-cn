## 文档加载器

### 腾讯云对象存储 (COS)

>[腾讯云对象存储 (COS)](https://www.tencentcloud.com/products/cos) 是一种分布式存储服务，允许您通过 HTTP/HTTPS 协议从任何位置存储任意数量的数据。
> `COS` 对数据结构或格式没有限制。它也没有存储桶大小限制和分区管理，几乎适用于任何用例，例如数据交付、数据处理和数据湖。
> COS 提供基于 Web 的控制台、多语言 SDK 和 API、命令行工具以及图形化工具。它与 Amazon S3 API 兼容良好，使您可以快速访问社区工具和插件。

安装 Python SDK：

<CodeGroup>
```bash pip
pip install cos-python-sdk-v5
```

```bash uv
uv add cos-python-sdk-v5
```
</CodeGroup>

## 向量存储

### 腾讯云向量数据库

>[腾讯云向量数据库](https://www.tencentcloud.com/products/vdb) 是一个全托管、自研的企业级分布式数据库服务，
> 专用于存储、检索和分析多维向量数据。该数据库支持多种索引类型和相似度计算方法，单个索引支持 10 亿向量、百万级 QPS 和毫秒级查询延迟。
> `腾讯云向量数据库` 不仅可以为大型模型提供外部知识库，提高大模型回答的准确性，还可以广泛应用于推荐系统、NLP 服务、计算机视觉和智能客服等 AI 领域。

安装 Python SDK：

<CodeGroup>
```bash pip
pip install tcvectordb
```

```bash uv
uv add tcvectordb
```
</CodeGroup>
