所有与 WRITER 相关的功能

> 本页介绍如何在 LangChain 中使用 [WRITER](https://writer.com/) 生态系统。更多信息请参阅 Writer [文档](https://dev.writer.com/home/introduction)。
> [Palmyra](https://writer.com/blog/palmyra/) 是由 `WRITER` 开发的大型语言模型 (LLM)。
>
> [Writer API](https://dev.writer.com/api-guides/introduction) 由一系列具有不同能力和价格点的 Palmyra 子模型提供支持。

## 安装与设置

通过以下命令安装集成包：
<CodeGroup>
```bash pip
pip install langchain-writer
```

```bash uv
uv add langchain-writer
```
</CodeGroup>

获取 WRITER API 密钥并将其设置为环境变量 (`WRITER_API_KEY`)

**弃用通知**：PDF 解析工具已弃用，将于 **2025 年 12 月 22 日** 移除。

**迁移路径**：我们计划为聊天补全功能引入一个预构建的 PDF 解析工具，该工具将提供类似的功能。此工具的工作方式将与其他预构建工具类似。当此替代方案可用时，我们将提供更多详细信息。
</Warning>
