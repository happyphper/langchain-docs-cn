本页面介绍如何使用 [Yeager.ai](https://yeager.ai) 生成 LangChain 工具和智能体。

## 什么是 Yeager.ai？
Yeager.ai 是一个旨在简化创建 AI 智能体和工具过程的生态系统。

其核心是 yAgents，一个无代码的 LangChain 智能体构建器，使用户能够轻松构建、测试和部署 AI 解决方案。yAgents 利用 LangChain 框架，可以无缝集成各种语言模型和资源，使其适用于开发者、研究人员和 AI 爱好者，应用于多种场景。

## yAgents
yAgents 是一个低代码生成式智能体，旨在帮助您轻松构建、原型设计和部署 LangChain 工具。

### 如何使用？
<CodeGroup>
```bash pip
pip install yeagerai-agent
yeagerai-agent
```

```bash uv
uv add yeagerai-agent
yeagerai-agent
```
</CodeGroup>

这将安装必要的依赖项并在您的系统上设置 yAgents。首次运行后，yAgents 会创建一个 .env 文件，您可以在其中输入您的 OpenAI API 密钥。您也可以在 Gradio 界面的 "Settings" 选项卡下直接进行此操作。

`OPENAI_API_KEY=<your_openai_api_key_here>`

我们推荐使用 GPT-4。不过，如果问题被充分分解，该工具也可以与 GPT-3 配合使用。

### 使用 yAgents 创建和执行工具
yAgents 使得创建和执行 AI 驱动的工具变得非常简单。以下是该过程的简要概述：
1.  **创建工具**：要向 yAgents 创建工具，请提供一个自然语言提示。提示应清晰描述工具的目的和功能。例如：
    `create a tool that returns the n-th prime number`

2.  **将工具加载到工具包中**：要将工具加载到 yAgents 中，只需向 yAgents 提供相应的命令即可。例如：
    `load the tool that you just created it into your toolkit`

3.  **执行工具**：要运行工具或智能体，只需向 yAgents 提供包含工具名称和任何所需参数的命令。例如：
    `generate the 50th prime number`

您可以在此处观看其工作原理的视频 [here](https://www.youtube.com/watch?v=KA5hCM3RaWE)。

随着您对 yAgents 越来越熟悉，您可以创建更高级的工具和智能体来自动化您的工作并提高生产力。

更多信息，请参阅 [yAgents 的 GitHub](https://github.com/yeagerai/yeagerai-agent) 或我们的 [文档](https://yeagerai.gitbook.io/docs/general/welcome-to-yeager.ai)。
