>[Embedchain](https://github.com/embedchain/embedchain) 是一个用于创建数据管道的 RAG 框架。它可以加载、索引、检索和同步所有数据。
>
>它提供 [开源包](https://github.com/embedchain/embedchain) 和 [托管平台解决方案](https://app.embedchain.ai/) 两种形式。

本笔记本展示了如何使用基于 `Embedchain` 的检索器。

# 安装

首先，你需要安装 [`embedchain` 包](https://pypi.org/project/embedchain/)。

你可以通过运行以下命令来安装：

```python
pip install -qU  embedchain
```

# 创建新的检索器

`EmbedchainRetriever` 有一个静态的 `.create()` 工厂方法，它接受以下参数：

* `yaml_path: string` 可选 -- YAML 配置文件的路径。如果未提供，则使用默认配置。你可以浏览 [文档](https://docs.embedchain.ai/) 来探索各种自定义选项。

```python
# 设置 API 密钥

import os
from getpass import getpass

if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = getpass()
```

```python
from langchain_community.retrievers import EmbedchainRetriever

# 使用默认选项创建一个检索器
retriever = EmbedchainRetriever.create()

# 或者，如果你想自定义，请传递 yaml 配置文件路径
# retriever = EmbedchainRetiever.create(yaml_path="config.yaml")
```

# 添加数据

在 embedchain 中，你可以添加尽可能多的支持的数据类型。你可以浏览我们的 [文档](https://docs.embedchain.ai/) 查看支持的数据类型。

Embedchain 会自动推断数据的类型。因此，你可以添加字符串、URL 或本地文件路径。

```python
retriever.add_texts(
    [
        "https://en.wikipedia.org/wiki/Elon_Musk",
        "https://www.forbes.com/profile/elon-musk",
        "https://www.youtube.com/watch?v=RcYjXbSJBN8",
    ]
)
```
