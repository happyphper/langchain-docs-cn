[GPT4All](https://gpt4all.io/index.html) 是一个免费使用、本地运行、注重隐私的聊天机器人。它不需要 GPU 或互联网连接。它包含流行的模型及其自有模型，例如 GPT4All Falcon、Wizard 等。

本笔记本解释了如何在 LangChain 中使用 [GPT4All 嵌入](https://docs.gpt4all.io/gpt4all_python_embedding.html#gpt4all.gpt4all.Embed4All)。

## 安装 GPT4All 的 Python 绑定

```python
pip install -qU  gpt4all > /dev/null
```

```python
from langchain_community.embeddings import GPT4AllEmbeddings
```

```python
gpt4all_embd = GPT4AllEmbeddings()
```

```text
Model downloaded at:  /Users/rlm/.cache/gpt4all/ggml-all-MiniLM-L6-v2-f16.bin
```

```python
text = "This is a test document."
```

## 嵌入文本数据

```python
query_result = gpt4all_embd.embed_query(text)
```

使用 `embed_documents` 方法可以嵌入多段文本。您还可以使用 [Nomic's Atlas](https://docs.nomic.ai/index.html) 将这些嵌入映射起来，以查看数据的可视化表示。

```python
doc_result = gpt4all_embd.embed_documents([text])
```
