>[TensorFlow Hub](https://www.tensorflow.org/hub) 是一个预训练机器学习模型的存储库，这些模型已准备好进行微调并可部署在任何地方。只需几行代码即可复用像 `BERT` 和 `Faster R-CNN` 这样的预训练模型。
>
>
让我们加载 TensorflowHub Embedding 类。

```python
from langchain_community.embeddings import TensorflowHubEmbeddings
```

```python
embeddings = TensorflowHubEmbeddings()
```
