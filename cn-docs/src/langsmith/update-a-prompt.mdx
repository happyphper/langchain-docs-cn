sidebarTitle: Update a prompt

在左侧边栏中导航到相应部分，或从应用程序主页点击您想要编辑的提示词。

## 更新元数据

要更新提示词的元数据（描述、用例等），请点击“编辑”铅笔图标。

![元数据编辑按钮](/langsmith/images/metadata-edit-button.png)

保存后，您的提示词元数据将被更新。

![更新提示词表单](/langsmith/images/update-prompt-form.png)

## 更新提示词内容

要更新提示词内容本身，您需要进入提示词游乐场。点击“在游乐场中编辑”。现在您可以对提示词进行修改，并使用不同的输入进行测试。当您对提示词满意时，点击“提交”以保存。

![在游乐场中编辑](/langsmith/images/edit-in-playground.png)
![提示词游乐场编辑提交](/langsmith/images/prompt-playground-edit-commit.png)

## 为提示词创建版本

当您向提示词添加提交时，会创建该提示词的一个新版本。您可以通过点击提示词视图中的“提交”选项卡来查看所有历史版本。

![提示词提交选项卡](/langsmith/images/prompt-commits-tab.png)
