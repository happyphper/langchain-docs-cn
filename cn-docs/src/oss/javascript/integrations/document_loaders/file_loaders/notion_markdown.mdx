---
title: Notion Markdown 导出
---
本示例演示了如何从 Notion 仪表板导出的页面中加载数据。

首先，按照官方说明[此处](https://www.notion.so/help/export-your-content)将您的 Notion 页面导出为 **Markdown 和 CSV** 格式。请确保选择 `包含子页面` 和 `为子页面创建文件夹`。

然后，解压下载的文件，并将解压后的文件夹移动到您的代码仓库中。该文件夹应包含您页面的 Markdown 文件。

将文件夹放入仓库后，只需运行以下示例：

```typescript
import { NotionLoader } from "@langchain/community/document_loaders/fs/notion";

export const run = async () => {
  /** 提供您的 Notion 文件夹的目录路径 */
  const directoryPath = "Notion_DB";
  const loader = new NotionLoader(directoryPath);
  const docs = await loader.load();
  console.log({ docs });
};
```
