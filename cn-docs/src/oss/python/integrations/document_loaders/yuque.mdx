title: 语雀
---
>[语雀](https://www.yuque.com/) 是一个专业的云端知识库，用于团队文档协作。

本笔记本介绍了如何从 `Yuque` 加载文档。

您可以在 [个人设置](https://www.yuque.com/settings/tokens) 页面点击个人头像来获取个人访问令牌。

```python
from langchain_community.document_loaders import YuqueLoader
```

```python
loader = YuqueLoader(access_token="<your_personal_access_token>")
```

```python
docs = loader.load()
```
