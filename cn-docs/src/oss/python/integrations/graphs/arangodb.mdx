>[ArangoDB](https://github.com/arangodb/arangodb) 是一个可扩展的图数据库系统，旨在更快地从互联数据中获取价值。它通过单一查询语言提供原生图、集成搜索引擎和 JSON 支持。`ArangoDB` 可在本地或云端运行。

本笔记本展示了如何使用 LLM 为 [ArangoDB](https://github.com/arangodb/arangodb#readme) 数据库提供自然语言接口。

## 环境设置

您可以通过 [ArangoDB Docker 镜像](https://hub.docker.com/_/arangodb) 运行一个本地的 `ArangoDB` 实例：

```
docker run -p 8529:8529 -e ARANGO_ROOT_PASSWORD= arangodb/arangodb
```

另一种方法是使用 [ArangoDB 云连接器包](https://github.com/arangodb/adb-cloud-connector#readme) 来运行一个临时的云实例：

```shell
pip install -qU  python-arango # ArangoDB Python 驱动
pip install -qU  adb-cloud-connector # ArangoDB 云实例配置器
pip install -qU  langchain-openai
pip install -qU  langchain
```

```python
# 实例化 ArangoDB 数据库
import json

from adb_cloud_connector import get_temp_credentials
from arango import ArangoClient

con = get_temp_credentials()

db = ArangoClient(hosts=con["url"]).db(
    con["dbName"], con["username"], con["password"], verify=True
)

print(json.dumps(con, indent=2))
```

if db.has_graph("GameOfThrones"):

## 查询 ArangoDB 数据库

我们现在可以使用 `ArangoDB Graph` QA 链来查询我们的数据。

```python
import os

os.environ["OPENAI_API_KEY"] = "your-key-here"
```

```python
from langchain_classic.chains import ArangoGraphQAChain
from langchain_openai import ChatOpenAI

chain = ArangoGraphQAChain.from_llm(
    ChatOpenAI(temperature=0), graph=graph, verbose=True
)
```

```python
chain.run("Is Ned Stark alive?")
```

```text
> 进入新的 ArangoGraphQAChain 链...

WITH Characters
FOR character IN Characters
FILTER character.name == "Ned" AND character.surname == "Stark"
RETURN character.alive

AQL 结果:
[True]

> 链结束。
```

## 链修改器

您可以修改以下 `ArangoDBGraphQAChain` 类变量的值，以改变链结果的行为。

```python
# 指定要返回的 AQL 查询结果的最大数量
chain.top_k = 10

# 指定是否在输出字典中返回 AQL 查询
chain.return_aql_query = True

# 指定是否在输出字典中返回 AQL JSON 结果
chain.return_aql_result = True

# 指定应进行的 AQL 生成尝试的最大次数
chain.max_aql_generation_attempts = 5

# 指定一组 AQL 查询示例，这些示例将传递给
# AQL 生成提示模板以促进小样本学习。
# 默认为空字符串。
chain.aql_examples = """
# Is Ned Stark alive?
RETURN DOCUMENT('Characters/NedStark').alive

# Is Arya Stark the child of Ned Stark?
FOR e IN ChildOf
    FILTER e._from == "Characters/AryaStark" AND e._to == "Characters/NedStark"
    RETURN e
"""
```

```python
chain.run("Is Ned Stark alive?")

# chain("Is Ned Stark alive?") # 返回一个包含 AQL 查询和 AQL 结果的字典
```

```text
> 进入新的 ArangoGraphQAChain 链...
AQL 查询 (1):
RETURN DOCUMENT('Characters/NedStark').alive

AQL 结果:
[True]

> 链结束。
```

```text
'是的，根据数据库中的信息，奈德·史塔克还活着。'
```
