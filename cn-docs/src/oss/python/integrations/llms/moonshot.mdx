[Moonshot](https://platform.moonshot.cn/) 是一家为企业和个人提供大语言模型服务的中国初创公司。

本示例将介绍如何使用 LangChain 与 Moonshot 进行交互。

```python
from langchain_community.llms.moonshot import Moonshot
```

```python
import os

# 从以下地址生成你的 API 密钥：https://platform.moonshot.cn/console/api-keys
os.environ["MOONSHOT_API_KEY"] = "MOONSHOT_API_KEY"
```

```python
llm = Moonshot()
# 或者使用特定模型
# 可用模型列表：https://platform.moonshot.cn/docs
# llm = Moonshot(model="moonshot-v1-128k")
```

```python
# 向模型提问
llm.invoke("What is the difference between panda and bear?")
```
