title: Featherless AI
---
[Featherless AI](https://featherless.ai/) 是一个无服务器 AI 推理平台，提供对超过 4300 多个开源模型的访问。我们的目标是让所有 AI 模型都能用于无服务器推理。我们通过 API 为不断扩展的开源权重模型库提供推理服务。

# 安装与设置
`pip install langchain-featherless-ai`
1. 在 [Featherless](https://featherless.ai/register) 注册一个账户。
2. 订阅一个计划，并从 [API Keys](https://featherless.ai/account/api-keys) 获取您的 API 密钥。
3. 将您的 API 密钥设置为环境变量 (`FEATHERLESSAI_API_KEY`)。

# 模型目录
访问我们的模型目录以查看所有模型的概览：https://featherless.ai/models
