>[FireCrawl](https://firecrawl.dev/?ref=langchain) 能够抓取任何网站并将其转换为可供大语言模型（LLM）使用的数据。
> 它会抓取所有可访问的子页面，并为每个页面提供干净的 Markdown 格式内容和元数据。无需站点地图。

## 安装与设置

安装 Python SDK：

<CodeGroup>
```bash pip
pip install firecrawl-py==0.0.20
```

```bash uv
uv add "firecrawl-py==0.0.20"
```
</CodeGroup>

## 文档加载器

查看[使用示例](/oss/integrations/document_loaders/firecrawl)。

```python
from langchain_community.document_loaders import FireCrawlLoader
```
