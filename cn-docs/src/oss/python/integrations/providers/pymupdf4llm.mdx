---
title: PyMuPDF4LLM
---
[PyMuPDF4LLM](https://pymupdf.readthedocs.io/en/latest/pymupdf4llm) 旨在简化为 LLM 和 RAG 应用提取 Markdown 格式 PDF 内容的过程。

[langchain-pymupdf4llm](https://github.com/lakinduboteju/langchain-pymupdf4llm) 将 PyMuPDF4LLM 集成到 LangChain 中，作为一个文档加载器。

```python
pip install -qU langchain-pymupdf4llm
```

```python
from langchain_pymupdf4llm import PyMuPDF4LLMLoader, PyMuPDF4LLMParser
```
