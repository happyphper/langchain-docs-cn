title: SemaDB
---
>[SemaDB](https://semafind.com/) 是一款简洁易用的向量相似性搜索引擎。它提供了一个低成本的云端托管版本，帮助您轻松构建 AI 应用。

通过我们的托管版本 SemaDB Cloud，"简洁易用"意味着无需计算 Pod 大小、无需定义模式、无需设置分区、无需调优参数、无需调整搜索算法、无需复杂安装、无需复杂 API。它已与 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 集成，提供透明的计费、自动分片以及交互式 API 测试平台。

## 安装

无需安装，可直接通过 [RapidAPI](https://rapidapi.com/semafind-semadb/api/semadb) 开始使用 SemaDB Cloud。

## 向量存储

`SemaDB` 集合有一个基础封装器，允许您将其用作向量存储。

```python
from langchain_community.vectorstores import SemaDB
```

您可以按照[此笔记本](/oss/integrations/vectorstores/semadb)中的教程学习如何使用该封装器。
