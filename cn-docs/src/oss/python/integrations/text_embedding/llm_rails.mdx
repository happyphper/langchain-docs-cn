title: LLMRails
---
让我们加载 LLMRails Embeddings 类。

要使用 LLMRails 嵌入，你需要通过参数传递 API 密钥，或者在环境中使用 `LLM_RAILS_API_KEY` 键来设置它。
要获取 API 密钥，你需要在 [console.llmrails.com/signup](https://console.llmrails.com/signup) 注册，然后前往 [console.llmrails.com/api-keys](https://console.llmrails.com/api-keys)，在平台上创建一个密钥后从那里复制。

```python
from langchain_community.embeddings import LLMRailsEmbeddings
```

```python
embeddings = LLMRailsEmbeddings(model="embedding-english-v1")  # 或者 embedding-multi-v1
```

```python
text = "This is a test document."
```

要生成嵌入向量，你可以查询单个文本，也可以查询一个文本列表。

```python
query_result = embeddings.embed_query(text)
query_result[:5]
```


```python
doc_result = embeddings.embed_documents([text])
doc_result[0][:5]
```
