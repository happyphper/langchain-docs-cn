title: 数据先驱
---
本笔记本将介绍如何使用 Dataherald 组件。

首先，你需要设置你的 Dataherald 账户并获取 API KEY：

1.  前往 dataherald 并在此处注册 [here](https://www.dataherald.com/)
2.  登录你的 Admin Console 后，创建一个 API KEY
3.  通过 pip 安装 dataherald

然后，我们需要设置一些环境变量：

1.  将你的 API KEY 保存到 DATAHERALD_API_KEY 环境变量中

```python
pip install dataherald
pip install -qU langchain-community
```

```python
import os

os.environ["DATAHERALD_API_KEY"] = ""
```

```python
from langchain_community.utilities.dataherald import DataheraldAPIWrapper
```

```python
dataherald = DataheraldAPIWrapper(db_connection_id="65fb766367dd22c99ce1a12d")
```

```python
dataherald.run("How many employees are in the company?")
```

```text
'select COUNT(*) from employees'
```
