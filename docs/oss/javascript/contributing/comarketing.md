LangChain 每月下载量超过 6000 万次，拥有大量构建 LLM 应用程序的开发者受众。除了仅仅列出集成，我们的目标是突出高质量、具有教育意义的示例，以启发开发者并推动生态系统发展。

<Note>

虽然我们偶尔会分享集成，但我们优先考虑提供有意义见解和最佳实践的内容。我们的主要社交渠道是 [Twitter](https://x.com/LangChain) 和 [LinkedIn](https://www.linkedin.com/company/langchain/)，我们在那里重点展示最佳示例。

</Note>

### 我们乐于推广的内容

:::: details <Icon icon="graduation-cap" style="margin-right: 8px; vertical-align: middle;" /> 教育内容

展示教育内容的博客、YouTube 视频和其他媒体。请注意，我们更倾向于内容不是以“这是如何使用集成 XYZ”为框架，而是“这是如何实现 ABC”，因为我们发现这对开发者更具教育意义和帮助。

::::

:::: details <Icon icon="cube" style="margin-right: 8px; vertical-align: middle;" /> 端到端应用程序

端到端应用程序对于希望构建应用的开发者来说是极好的资源。我们更倾向于突出本质上更复杂/更具智能体（agentic）特性的应用程序，并且使用 [LangGraph](https://github.com/langchain-ai/langgraphjs) 作为编排框架。我们对涉及以下任何方面的内容特别感兴趣：

- 长期记忆（memory）系统
- 人机协同（human-in-the-loop）交互模式
- 多智能体（multi-agent）架构

::::

我们乐于突出新颖的研究！无论是基于 LangChain 构建的研究还是与之集成的研究。
