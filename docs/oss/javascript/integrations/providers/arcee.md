>[Arcee](https://www.arcee.ai/about/about-us) 致力于开发和推进我们称之为 SLM 的模型——即小型化、专业化、安全且可扩展的语言模型。
> 通过提供 SLM 适配系统和无缝、安全的集成，
> `Arcee` 使企业能够充分利用领域适配语言模型的全部潜力，
> 从而推动运营中的变革性创新。

## 安装与设置

获取您的 `Arcee API` 密钥。

## LLMs

查看[使用示例](/oss/javascript/integrations/llms/arcee)。

```python
from langchain_community.llms import Arcee
```
