>[CLOVA Studio](https://api.ncloud-docs.com/docs/ai-naver-clovastudio-summary) 是 [Naver Cloud Platform](https://www.ncloud.com/) 提供的一项服务，它利用 `HyperCLOVA` 语言模型（一种超大规模 AI 技术），根据用户输入输出通过 AI 技术生成的短语。

## 嵌入模型

请参阅[安装说明和使用示例](/oss/javascript/integrations/text_embedding/clova)。

from langchain_community.embeddings import ClovaEmbeddings
