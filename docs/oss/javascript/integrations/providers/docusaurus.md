>[Docusaurus](https://docusaurus.io/) 是一个静态站点生成器，它提供了开箱即用的文档功能。

## 安装与设置

::: code-group

```bash [pip]
pip install -U beautifulsoup4 lxml
```

```bash [uv]
uv add beautifulsoup4 lxml
```

:::

## 文档加载器
