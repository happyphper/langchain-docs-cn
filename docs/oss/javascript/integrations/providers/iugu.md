>[Iugu](https://www.iugu.com/) 是一家巴西的服务和软件即服务（SaaS）公司。它为电子商务网站和移动应用程序提供支付处理软件和应用程序编程接口。

## 安装与设置

`Iugu API` 需要一个访问令牌，该令牌可在 `Iugu` 仪表板中找到。
