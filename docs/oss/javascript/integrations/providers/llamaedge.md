>[LlamaEdge](https://llamaedge.com/docs/intro/) 是在本地或边缘设备上运行定制化和微调大语言模型（LLM）的最简单、最快速的方式。

* 轻量级推理应用。`LlamaEdge` 的体积是 MB 级别，而非 GB 级别。
* 原生且支持 GPU 加速的性能。
* 支持多种 GPU 和硬件加速器。
* 支持多种优化的推理库。
* 提供广泛的 AI / LLM 模型选择。

## 安装与设置

请参阅[安装说明](https://llamaedge.com/docs/user-guide/quick-start-command)。
