[Nimble](https://www.linkedin.com/company/nimbledata) 是首个商业外部数据平台，凭借我们屡获殊荣的 AI 驱动数据结构化技术，让数据决策变得前所未有的简单。Nimble 将商业用户与公共网络知识连接起来。

我们为企业提供关键任务的实时外部数据，以解锁高级商业智能、价格比较以及销售和营销所需的其他公共数据。我们将数据转化为即时的商业价值。

如果您想了解更多关于 Nimble 的信息，请访问我们的网站 [nimbleway.com](https://www.nimbleway.com/)。

## 检索器

### NimbleSearchRetriever

使开发者能够构建 RAG 应用程序和 AI 智能体，这些程序可以从网络上的任何地方搜索、访问和检索在线信息。

我们需要安装 `langchain-nimble` Python 包。

```python
pip install -U langchain-nimble
```

查看[使用示例](/oss/javascript/integrations/retrievers/nimble/)。

```python
from langchain_nimble import NimbeSearchRetriever
```

请注意，需要进行身份验证，请参阅文档中的[设置部分](/oss/javascript/integrations/retrievers/nimble/#setup)。
