[Pebblo](https://www.daxa.ai/pebblo) 使开发者能够安全地加载和检索数据，以将其生成式 AI 应用推进到部署阶段，而无需担心组织的合规性和安全性要求。Pebblo SafeLoader 能识别加载数据中发现的语义主题和实体，而 Pebblo SafeRetriever 则对检索到的上下文强制执行身份和语义控制。结果会在 UI 界面或 PDF 报告中汇总。

## Pebblo 概述

`Pebblo` 为生成式 AI 应用程序提供了一种安全的数据加载和检索方式。
它包括：

1.  **具备身份感知能力的 Safe Loader**，用于加载数据并识别语义主题和实体。
2.  **SafeRetrieval**，对检索到的上下文强制执行身份和语义控制。
3.  **用户数据报告**，汇总已加载和检索的数据。
