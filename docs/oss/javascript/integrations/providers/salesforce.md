[Salesforce](https://www.salesforce.com/) 是一家基于云的软件公司，提供客户关系管理（CRM）解决方案以及一套专注于销售、客户服务、营销自动化和分析的企业应用程序。

[langchain-salesforce](https://pypi.org/project/langchain-salesforce/) 实现了使 LLM 能够与 Salesforce 数据交互的工具。

## 安装与设置

::: code-group

```bash [pip]
pip install langchain-salesforce
```

```bash [uv]
uv add langchain-salesforce
```

:::
