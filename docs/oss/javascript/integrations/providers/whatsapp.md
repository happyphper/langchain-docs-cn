>[WhatsApp](https://www.whatsapp.com/)（也称为 `WhatsApp Messenger`）是一款免费、跨平台、集中式的即时通讯（IM）和互联网语音协议（VoIP）服务。它允许用户发送文本和语音消息，进行语音和视频通话，以及分享图片、文档、用户位置和其他内容。

## 安装与设置

无需任何特殊设置。

查看[使用示例](/oss/integrations/document_loaders/whatsapp_chat)。

from langchain_community.document_loaders import WhatsAppChatLoader
