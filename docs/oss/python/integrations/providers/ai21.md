>[AI21 Labs](https://www.ai21.com/about) 是一家专注于自然语言处理（NLP）的公司，致力于开发能够理解和生成自然语言的人工智能系统。

本页介绍了如何在 `LangChain` 中使用 `AI21` 生态系统。

## 安装与设置

- 获取一个 AI21 API 密钥，并将其设置为环境变量 (`AI21_API_KEY`)
- 安装 Python 包：

::: code-group

```bash [pip]
pip install langchain-ai21
```

```bash [uv]
uv add langchain-ai21
```

:::

## 聊天模型

### AI21 Chat

查看[使用示例](/oss/python/integrations/chat/ai21)。

```python
from langchain_ai21 import ChatAI21
```
