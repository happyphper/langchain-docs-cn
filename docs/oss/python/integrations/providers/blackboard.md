>[Blackboard Learn](https://en.wikipedia.org/wiki/Blackboard_Learn)（前身为 `Blackboard Learning Management System`）
> 是一个基于网络的虚拟学习环境和学习管理系统，由 Blackboard Inc. 开发。
> 该软件具有课程管理、可定制的开放架构和可扩展的设计，允许
> 与学生信息系统和身份验证协议集成。它可以安装在本地服务器上，
> 由 `Blackboard ASP Solutions` 托管，或作为托管在 Amazon Web Services 上的软件即服务提供。
> 其主要目的包括为传统面对面授课的课程增加在线元素，
> 以及开发完全在线、极少或无需面对面会议的课程。

## 安装与设置

它没有任何特殊的设置要求。
