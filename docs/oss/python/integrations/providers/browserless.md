>[Browserless](https://www.browserless.io/docs/start) 是一项服务，允许您在云端运行无头 Chrome 实例。这是一种大规模运行基于浏览器的自动化的绝佳方式，无需担心管理自己的基础设施。

## 安装与设置

我们需要在此处获取 [API 密钥](https://www.browserless.io/pricing/)。
