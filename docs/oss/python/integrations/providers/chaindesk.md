>[Chaindesk](https://chaindesk.ai) 是一个[开源](https://github.com/gmpetrov/databerry)文档检索平台，旨在帮助你将个人数据与大型语言模型连接起来。

## 安装与设置

我们需要注册 Chaindesk，创建一个数据存储，添加一些数据并获取你的数据存储 API 端点 URL。
我们需要 [API 密钥](https://docs.chaindesk.ai/api-reference/authentication)。
