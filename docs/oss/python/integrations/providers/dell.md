戴尔是一家全球性的科技公司，提供包括人工智能解决方案在内的各类硬件、软件和服务。其人工智能产品组合包含专为人工智能工作负载构建的基础设施，其中包括为人工智能数据管理优化的戴尔 PowerScale 存储系统。

## PowerScale

戴尔 [PowerScale](https://www.dell.com/en-us/shop/powerscale-family/sf/powerscale) 是一款企业级横向扩展存储系统，它搭载了业界领先的 OneFS 文件系统，可以部署在本地或云端。

### 安装与设置

::: code-group

```bash [pip]
pip install powerscale-rag-connector
```

```bash [uv]
uv add powerscale-rag-connector
```

:::
