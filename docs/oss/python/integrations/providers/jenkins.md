[Jenkins](https://www.jenkins.io/) 是一个开源自动化平台，它帮助软件团队简化其开发工作流程。在 DevOps 社区中，它被广泛采用，作为通过 CI/CD 流水线自动化构建、测试和部署应用程序的工具。

## 安装与设置

::: code-group

```bash [pip]
pip install langchain-jenkins
```

```bash [uv]
uv add langchain-jenkins
```

:::
