>[MongoDB](https://www.mongodb.com/) 是一个面向文档的 NoSQL 数据库，支持具有动态模式的类 JSON 文档。

**注意：**
- 有关其他 `MongoDB` 集成，请参阅 [MongoDB Atlas 页面](/oss/integrations/providers/mongodb_atlas)。

## 安装与设置

安装 Python 包：

::: code-group

```bash [pip]
pip install langchain-mongodb
```

```bash [uv]
uv add langchain-mongodb
```

:::
