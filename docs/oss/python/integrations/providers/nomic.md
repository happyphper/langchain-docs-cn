title: Nomic
---
>[Nomic](https://www.nomic.ai/) 致力于构建工具，使每个人都能与 AI 规模的数据集交互，并在消费级计算机上运行 AI 模型。

`Nomic` 目前提供两款产品：

- `Atlas`：可视化数据引擎
- `GPT4All`：开源边缘语言模型生态系统

Nomic 集成存在于两个合作伙伴包中：[langchain-nomic](https://pypi.org/project/langchain-nomic/) 和 [langchain-community](https://pypi.org/project/langchain-community/)。

## 安装

您可以通过以下方式安装它们：

::: code-group

```bash [pip]
pip install -U langchain-nomic
pip install -U langchain-community
```

```bash [uv]
uv add langchain-nomic
uv add langchain-community
```

:::

## 大语言模型

### GPT4All

查看[使用示例](/oss/python/integrations/llms/gpt4all)。

```python
from langchain_community.llms import GPT4All
```
