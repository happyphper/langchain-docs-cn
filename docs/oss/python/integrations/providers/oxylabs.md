title: Oxylabs
---
[Oxylabs](https://oxylabs.io/) 是一个市场领先的网络情报收集平台，遵循最高的商业、道德和合规标准，助力全球企业解锁数据驱动的洞察。

[langchain-oxylabs](https://pypi.org/project/langchain-oxylabs/) 实现了相关工具，使 LLMs 能够与 Oxylabs Web Scraper API 进行交互。

## 安装与设置

::: code-group

```bash [pip]
pip install langchain-oxylabs
```

```bash [uv]
uv add langchain-oxylabs
```

:::
