>[PygmalionAI](https://pygmalion.chat/) 是一家支持开源模型的公司，通过为 [Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine) 提供推理端点服务来实现。

## 安装与设置

::: code-group

```bash [pip]
pip install aphrodite-engine
```

```bash [uv]
uv add aphrodite-engine
```

:::
