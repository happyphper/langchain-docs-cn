>[01.AI](https://www.lingyiwanwu.com/en) 由李开复博士创立，是一家处于 AI 2.0 前沿的全球化公司。他们提供尖端的大型语言模型，包括参数规模从 6B 到数千亿不等的 Yi 系列模型。01.AI 还提供多模态模型、开放的 API 平台以及像 Yi-34B/9B/6B 和 Yi-VL 这样的开源选项。

## 安装与设置

请注册并从中国站点[此处](https://platform.lingyiwanwu.com/apikeys)或全球站点[此处](https://platform.01.ai/apikeys)获取 API 密钥。

## LLMs

查看[使用示例](/oss/python/integrations/llms/yi)。

```python
from langchain_community.llms import YiLLM
```
