与 LangChain 社区建立联系，获取学习资源，并获得构建应用所需的支持。

## 学习资源

通过我们全面的学习材料，开启您的旅程或深化您的知识。

- **[Chat LangChain](https://chat.langchain.com/)**: 向文档询问任何关于 LangChain 的问题，由实时文档提供支持
- **[API 参考](https://reference.langchain.com/python/)**: 所有 LangChain 包的完整文档

## 社区支持

通过我们活跃的社区渠道，从其他开发者和 LangChain 团队获得帮助。

- **[社区论坛](https://forum.langchain.com/)**: 提问、分享解决方案并讨论最佳实践
- **[社区 Slack](https://www.langchain.com/join-community)**: 与其他构建者联系并获得快速帮助

## 专业支持

针对企业需求和关键应用，访问专属的支持渠道。

- **[支持门户](https://support.langchain.com/)**: 提交工单并跟踪支持请求
- **[LangSmith 状态](https://status.smith.langchain.com/)**: LangSmith 服务和 API 的实时状态

## 贡献

帮助我们为所有人改进 LangChain。无论是修复错误、添加功能还是改进文档，我们都欢迎您的贡献。

- **[贡献指南](/oss/python/contributing/overview)**: 关于为 LangChain 做出贡献所需了解的一切
