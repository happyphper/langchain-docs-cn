在 [LangSmith UI](https://smith.langchain.com) 中，请确保您的 OpenAI API 密钥已设置为 [工作区密钥](/langsmith/administration-overview#workspace-secrets)。

1.  导航至 <Icon icon="gear" /> **设置**，然后转到 **密钥** 选项卡。
1.  选择 **添加密钥**，输入 `OPENAI_API_KEY` 作为密钥，并将您的 API 密钥填入 **值** 字段。
1.  选择 **保存密钥**。

<Note>
在 LangSmith UI 中添加工作区密钥时，请确保密钥名称与您的模型提供商所期望的环境变量名称相匹配。
</Note>
